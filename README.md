# Audrey's TF2 Bot Detector List

A list of cheaters in Team Fortress 2, automatically generated from my own demos.

## What is this?

This list contains cheaters who have 10+ detections across 4400+ demos from between 2023-2026. It's automatically generated using a modified version of [this](https://github.com/MegaAntiCheat/analysis-template-rust). This should be pretty accurate, but keep in mind there might be very few false positives.
